Числа Фибоначчи
Использовались точки останова, которые позволяют приостановить выполнение программы в определенных местах, чтобы могли проанализировать состояние программы в этот момент.
Также шаг с обходом для передвижения по коду, изменения просматривались в окне контрольных значений для мониторинга переменных.

Галактика
Использовались точки останова с шагом для передвижения по коду, горячая перезагрузка для запуска изменений, которая позволяет выполнять программу пошагово, используя такие команды, как F10 (выполнить следующую строку) и F11 (шаг с переходом в функцию). Это помогает найти область кода, в которой возникает проблема.

Буквы
Точки останова с обходом и шагом, с шагом выхода для передвижения по коду. Изменения переменных просматривались наведением курсора на их названия, а также с помощью окон. Контроль вызываемых функций с помощью окна Стека Вызовов для понимания того, какие функции были вызваны и в каком порядке, что помогает выявить ошибки в логике программы.
